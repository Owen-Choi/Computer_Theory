NULL 처리 함수

	데이터가 NULL인 경우 산술 연산자나 비교 연산자가 우리가 예상한 대로 동작하지 않는다.
	하지만 특정 열의 데이터가 NULL인 경우에 연산 수행을 위해 데이터를 NULL이 아닌 다른 값으로
	대체해 주어야 할 때가 종종 있다. 그럴때 NULL 처리 함수를 사용한다.

	NVL 함수 
		NVL([NULL인지 여부를 검사할 데이터 또는 열(필수)], 
		[앞의 데이터가 NULL일 경우 반환할 데이터(필수)])
	==> 입력한 열 또는 데이터가 NULL이 아닐 경우 데이터를 그대로 반환하고 NULL일 경우
		지정한 데이터를 반환한다.

	보통 SAL+NVL(COMM, 0) 등의 형식으로 많이 사용한다.
	(비어있는 데이터를 0으로 바꾸어 산술 연산)

	NVL2 함수
		NVL2([NULL인지 여부를 검사할 데이터 또는 문자열], 
		[앞 데이터가 NULL이 아닐 경우 반환할 데이터 또는 계산식(필수)],
		[앞 데이터가 NULL일 경우 반환할 데이터 또는 계산식(필수)])

	만약 추가수당(COMM)의 존재 여부 자체만을 놓고 볼 때, 추가수당이 존재한다면 O를, 그렇지 않다면 X를
	출력하고 싶을 경우 NVL2함수를 사용할 수 있다. 금액같은 민감한 데이터는 노출되지 않아야 되는 경우도
	있기 때문이다.

	또한 연봉을 계산할 때, 추가수당의 존재 여부에 따라 계산식이 달라질 수 있는데, 그 명령어는 다음과 같다.
		NVL2(COMM, SAL*12+COMM, SAL*12) AS ANNSAL
	추가수당이 존재한다면 월급에 12를 곱한 값 + 추가수당이 계산식이 되고, 추가수당이 없다면
	월급에 12를 곱한 값이 계산식이 된다.
